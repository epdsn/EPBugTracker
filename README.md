# EPBugTracker

EPBugTracker is a cross-platform bug tracking and review application built with .NET MAUI targeting .NET 10. The app helps teams manage, review, and track bug fixes across projects and platforms (Windows, macOS, iOS, Android).

Key features
- Create, edit and organize bug reports
- Assign bugs to team members and set priorities
- Review and comment on proposed fixes
- Attach screenshots and logs to bug reports
- Search, filter and tag issues
- Cross-platform UI using .NET MAUI

Getting started

Prerequisites
- .NET 10 SDK
- Visual Studio with .NET MAUI workload (Windows) or Visual Studio for Mac with MAUI support

Build and run
1. Open the solution in Visual Studio with the MAUI workload installed.
2. Select the target platform (Windows, Android, iOS, MacCatalyst) and run from the IDE.

Alternatively, build from the command line:

```bash
dotnet build
```

(For platform-specific runs use the IDE or platform-specific `dotnet` target framework options.)
